# NUTRI TRADE LTD — Лендинг B2B поставщика

Этот проект представляет собой структурированный, чистый и адаптивный одностраничный сайт для болгарской B2B-компании **NUTRI TRADE LTD**. Код оптимизирован, разбит по логическим блокам и очищен от лишних комментариев для удобства дальнейшей поддержки.

## Как открыть и запустить проект

Проект опубликован в сети и доступен для просмотра в любом браузере. 

### Вариант 1. Просмотр работающего сайта (GitHub Pages)
Для быстрого ознакомления с интерфейсом и тестирования функционала перейдите по прямой ссылке:
https://chatorsn.github.io/nutri-trade-landing/

### Вариант 2. Локальный запуск для разработки
Если вам необходимо открыть проект на компьютере локально:
1. Скачайте все файлы проекта (или клонируйте репозиторий) в одну папку на вашем устройстве.
2. Убедитесь, что файлы `index.html`, `style.css` и `script.js` находятся в корневой директории.
3. Дважды кликните по файлу `index.html`. Сайт откроется в вашем браузере по умолчанию.

Для полноценной работы и редактирования рекомендуется открывать папку с проектом через редактор кода (например, VS Code) с использованием расширения **Live Server**.

## Ключевой функционал и реализованные решения

* **B2B-форма обратной связи с валидацией:** Реализована проверка всех обязательных полей (включая чекбокс согласия с политикой). При успешном заполнении форма переходит в состояние `success` и выводит кастомное уведомление без перезагрузки страницы.
* **Интерактивный аккордеон в секции «Как мы работаем»:** Длинные шаги операционного цикла упакованы в раскрывающиеся блоки, что разгружает интерфейс и превращает страницу в аккуратный лендинг, а не массивный документ.
* **Модальные окна для юридических документов:** Тексты Privacy Policy и Cookies Policy встроены прямо в разметку и открываются в адаптивных модальных окнах, сохраняя доступность всей юридической информации.
* **Cookie-баннер:** Всплывающее уведомление внизу экрана с возможностью принять условия. Выбор пользователя фиксируется в `localStorage`, предотвращая повторное появление баннера при перезагрузке.
* **Полная адаптивность:** Сайт оптимизирован под любые разрешения экранов. Для мобильных устройств реализована удобная навигация через кнопку-гамбургер.

## Структура файлов проекта

* `index.html` — семантическая структура страницы, текстовый контент, формы обратной связи, куки-баннер и модальные окна юридических документов.
* `style.css` — упорядоченные стили, включая глобальные настройки, адаптивную сетку (Flexbox/Grid), эффекты стекломорфизма (Glassmorphism) и медиа-запросы для мобильных устройств.
* `script.js` — интерактивная логика сайта: мобильное меню, плавный скролл, работа аккордеона, управление cookie-баннером, открытие модальных окон политик конфиденциальности и фронтенд-валидация B2B-формы.
* `docs/` — папка со служебной документацией проекта:
  * `TASK.md` — исходное техническое задание.
  * `PROMPT_PLAN.md` — пошаговый план промптов, использованных при проектировании.
  * `AI_USAGE.md` — обязательный отчет об опыте взаимодействия с AI в процессе разработки.
