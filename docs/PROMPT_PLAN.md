# Пошаговый план разработки (Prompt Plan)

Разработка архитектуры, визуального стиля и интерактивного контента велась строго по этапам на основе документов `01-landing-client-content.md` и `01-landing-client-brief.docx`.

## Шаг 1: Анализ контента и разметка структуры (HTML)
- Проведен детальный анализ исходного клиентского текста. Выстроена логическая иерархия интерфейса, состоящая из 8 обязательных информационных блоков.
- Написана семантическая разметка `index.html` с использованием тегов `<header>`, `<main>`, `<section>`, `<form>`, `<footer >`.
- Интегрированы все текстовые блоки заказчика: спецификации калибров продукции, этапы операционного цикла и официальные болгарские юридические реквизиты компании.

## Шаг 2: Визуальное оформление и адаптивность (CSS)
- Реализован современный корпоративный дизайн с темным премиальным фоном и карточками в стиле «матового стекла» (эффект Glassmorphism через `backdrop-filter`).
- Оформлены списки требований торговых сетей и форматы фасовки сырья с использованием кастомных стилей и маркеров.
- Добавлена полноценная адаптивность (Flexbox/Grid) и медиа-запросы для корректного отображения структуры сайта на любых устройствах, включая мобильные телефоны.

## Шаг 3: Бизнес-логика, валидация и интерактивные UI-компоненты (JS)
- Создан изолированный файл `script.js` для управления всей интерактивной логикой сайта.
- Реализован перехват события `submit` для B2B-формы обратной связи: внедрена фронтенд-валидация обязательных полей, проверка e-mail регулярным выражением и блокировка отправки без согласия с политикой. Настроено состояние `success` со сбросом полей без перезагрузки страницы.
- Добавлен обработчик для плавного скролла от главной CTA-кнопки прямиком к форме запроса КП.

## Шаг 4: Развертывание юридического слоя и бонусов (GDPR & UX)
- **Модальные окна:** Вместо создания отдельных страниц, подробные тексты *Политики конфиденциальности* (Privacy Policy) и *Политики Cookies* встроены прямо в разметку и открываются в адаптивных модальных окнах при клике из футера.
- **Интерактивный аккордеон:** Сложный 6-этапный операционный цикл из Блока 4 упакован в раскрывающиеся блоки (аккордеон), что разгрузило страницу и сделало её удобной для чтения.
- **Cookie-баннер:** Добавлено всплывающее уведомление о куки, интегрированное с `localStorage` браузера, для сохранения выбора пользователя и предотвращения повторных показов баннера.
